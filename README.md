# Fastapi + Elm + Tailwindcss tech stack 

1. Setup elm 
    * clone the elm-vite-setup repo in the ui directory 
    * `git clone git@github.com:Farooq-azam-khan/elm-vite-setup.git ui`
2. Install npm packages
    * `cd ui`
    * `pnpm install` 
    * `pnpm start` - to start the local development server 
3. Install poetry 
    * ubuntu/debian based systems: `sudo apt install python3-poetry` 
4. Install dependencies
    * `poetry install` in the root folder 
5. Activate poetry environment 
    * `poetry shell`
6. Run the fastapi server 
    * `python main.py`
